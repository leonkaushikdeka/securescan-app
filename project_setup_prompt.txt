# Flutter Setup Prompt

**Copy and use this prompt before starting any Flutter Android project:**

---

I am creating a new Flutter project targeting Android. Before we begin, please ensure the following configurations are set up correctly to avoid common build issues:

## Environment Requirements
- Java 17+ (verify with `java -version`)
- Android SDK installed and updated
- Flutter SDK installed with PATH configured
- Android licenses accepted (`flutter doctor --android-licenses`)

## Project Configuration (Create with these settings)

### 1. android/settings.gradle
Use these plugin versions:
```
plugins {
    id "com.android.application" version "8.7.0" apply false
    id "org.jetbrains.kotlin.android" version "1.8.22" apply false
    id "dev.flutter.flutter-gradle-plugin" version "2.0.0" apply false
}
```

### 2. android/gradle/wrapper/gradle-wrapper.properties
```
distributionUrl=https\://services.gradle.org/distributions/gradle-8.9-all.zip
```

### 3. android/app/build.gradle
```
android {
    namespace = "com.example.appname"
    compileSdk = 36
    ndkVersion = "27.0.12077973"

    compileOptions {
        sourceCompatibility = JavaVersion.VERSION_17
        targetCompatibility = JavaVersion.VERSION_17
        coreLibraryDesugaringEnabled true
    }

    kotlinOptions {
        jvmTarget = JavaVersion.VERSION_17
    }

    defaultConfig {
        minSdk = 24
        targetSdk = 36
    }
}

dependencies {
    coreLibraryDesugaring 'com.android.tools:desugar_jdk_libs:2.1.3'
}
```

### 4. android/build.gradle
Ensure allprojects uses Google and Maven Central:
```
allprojects {
    repositories {
        google()
        mavenCentral()
    }
}
```

## Pre-Flight Checklist
Before running `flutter run`:
- [ ] `flutter doctor` shows no errors
- [ ] `flutter devices` shows emulator/device
- [ ] Run `flutter clean` before first build
- [ ] Run `flutter pub get` after any pubspec changes

## Common Fixes to Apply If Issues Occur
1. **AGP too old**: Update to 8.7.0 in settings.gradle
2. **Gradle too old**: Update to 8.9 in gradle-wrapper.properties
3. **NDK mismatch**: Set ndkVersion = "27.0.12077973"
4. **Core library desugaring**: Enable + add dependency
5. **Java version**: Set source/targetCompatibility to VERSION_17
