# SIMPLE PROJECT SETUP PROMPT
# Copy this and fill in the blanks before starting a new project

═══════════════════════════════════════════════════════════════
FLUTTER ANDROID PROJECT - VERSION-AWARE SETUP
═══════════════════════════════════════════════════════════════

STEP 1: CHECK CURRENT VERSIONS
Run these commands and note the results:
□ java -version
□ flutter doctor
□ flutter doctor -v | grep -A5 "Android"

STEP 2: GET LATEST COMPATIBLE VERSIONS
- Check AGP: https://developer.android.com/studio/releases/gradle
- Check Gradle: https://gradle.org/releases/
- Note versions here: AGP: _____ Gradle: _____ NDK: _____ Java: _____

STEP 3: CREATE PROJECT
```bash
flutter create --androidx myappname
cd myappname
```

STEP 4: UPDATE CONFIG FILES
File: android/settings.gradle
```gradle
plugins {
    id "com.android.application" version "8.7.0" apply false
    id "org.jetbrains.kotlin.android" version "1.8.22" apply false
    id "dev.flutter.flutter-gradle-plugin" version "2.0.0" apply false
}
```

File: android/gradle/wrapper/gradle-wrapper.properties
```properties
distributionUrl=https\://services.gradle.org/distributions/gradle-8.9-all.zip
```

File: android/app/build.gradle
```gradle
android {
    namespace = "com.example.myappname"
    compileSdk = 36
    ndkVersion = "27.0.12077973"

    compileOptions {
        sourceCompatibility = JavaVersion.VERSION_17
        targetCompatibility = JavaVersion.VERSION_17
        coreLibraryDesugaringEnabled true
    }

    kotlinOptions {
        jvmTarget = JavaVersion.VERSION_17
    }

    defaultConfig {
        minSdk = 24
        targetSdk = 36
    }
}

dependencies {
    coreLibraryDesugaring 'com.android.tools:desugar_jdk_libs:2.1.3'
}
```

STEP 5: BUILD & RUN
```bash
flutter clean
flutter pub get
flutter run -d emulator-5554
```
